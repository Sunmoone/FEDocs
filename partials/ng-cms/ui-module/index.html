模块与界面

AngularJS　作为数据框架，本身并不提供UI，但是其繁荣的生态系统提供了大量拓展。

AngularUI　提供了许多AngularJS可拓展的模块，一般将UI-Bootstrap模块作为CSS框架，而NG-Grid是一款强大的动态可操作表格模块。

  • 动态数据表格（提供工具栏的增删改查，分页，排序，统计）
  • 表单验证
  • 弹窗
  • 下拉导航、树形导航、侧导航
  • 日期时间选择器
  • Message 提示

以上等等，都可以通过http://angular-ui.github.io/获得。

如果涉及的UI不需要与AngularJS数据交互的话，也可以不通过注册模块的方式引入。
