媒体对象

这是一个抽象的样式，用以构建不同类型的组件，这些组件都具有在文本内容的左或右对齐的图片（就像blog内容或Tweets等）。

默认案例

默认样式是在内容区域的左或右侧浮动一个媒体对象（图片、视频、音频）。

Media heading
