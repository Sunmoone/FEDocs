CSS3 媒体查询

概述

通过媒体查询为不同的设备和屏幕大小适配不同的样式。

Media 代码
/* media */
/* 横屏 */
@media screen and (orientation:landscape){
      
}
/* 竖屏 */
@media screen and (orientation:portrait){
      
}
/* 窗口宽度 > 1440 */
@media screen and (min-width:1440px){
      
}				
/* 窗口宽度 < 1440,设计宽度=1280 */
@media screen and (max-width:1440px){
      
}				
/* 窗口宽度 < 1280,设计宽度=960 */
@media screen and (max-width:1279px){
      
}				
/* 窗口宽度 < 960,设计宽度=768 */
@media screen and (max-width:959px){
      
}
/* 窗口宽度 < 768,设计宽度=640 */
@media screen and (max-width:767px){
      
}
/* 窗口宽度 < 640,设计宽度=480 */
@media screen and (max-width:639px){
      
}
/* 窗口宽度 < 480,设计宽度=320 */
@media screen and (max-width:479px){
      
}
/* windows UI 贴靠 */
@media screen and (-ms-view-state:snapped){
      
}
/* 打印 */
@media print{
      
}
使用方法

通过右键另存为下载 reset.css，直接使用或将代码复制到你的css文件顶部。

使用举例

窗口宽度window 大于960px 小于1440px 时，设置#wrapper 宽度800px

窗口宽度window 小于960px 时， 设置#wrapper 宽度600px

/* 窗口宽度 < 1440px 时的样式规则 */
@media screen and (max-width:1440px){
   #wrapper{ width:800px } 
}
/* 窗口宽度 < 960px 时的样式规则 */
@media screen and (max-width:959px){
   #wrapper{ width:600px }   
}
