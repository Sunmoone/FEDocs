基础库
"基础库"是一些经常需要使用的CSS集合，作为(Public)公共基础可以被项目范围内引用，应该保持其唯一性，专人维护，不能随意修改。
布局
"布局"总结了一些常用的页面布局，两栏，三栏，满屏，自适应...另外还包含Bootstrap（源于Twitter的开源CSS框架）的栅格系统，固定布局，流式布局，响应式布局内容以及区别。
样式、组件、JavaScript插件
这三部分内容取自Bootstrap 第二版，包含几乎所有常用的页面元素，表格，表单，按钮，图标，Tab切换，下拉菜单，弹窗...
Bootstrap.css是由Less语言编译得到，通过Bootstrap.less文件可以很方便的批量修改样式，以形成不同风格的皮肤UI。
下载Bootstrap.css并引入页面，如果所需要构建的元素涉及到JavaScript，则还需要引入Bootstrap.js。或者下载完整源码包，定制更多功能。
