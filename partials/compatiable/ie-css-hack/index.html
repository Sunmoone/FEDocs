为IE 编写代码

1. IE条件注释

所有版本

 <!--[IF IE]>
     <link type=”text/css” href=”ie.css” rel=”stylesheet” />
 <![ENDIF]–>

指定版本

<!--[IF IE 6]>
     <link type=”text/css” href=”ie6.css” rel=”stylesheet” />
 <![ENDIF]-->

指定区间

<!--[IF lt IE 8]>
     <link type=”text/css” href=”ie8-.css” rel=”stylesheet” />
 <![ENDIF]-->

2. 样式属性前缀法

IE 6                  .test{_width:200px;}
IE6/IE7               .test{+width:100px;}
IE6/IE7               .test{*width:200px}

3. 选择符前缀法

IE6/IE7        *html div{border-color:red;}
IE 7           *+html div{border-color:red;}

4. 样式属性后缀法

all IE           .test{background:#ace\9}
gte IE8          .test{background:#ace\0}
gte IE9          .test{background:#ace\9\0}

5. :root根元素伪选择

:root是CSS3定义的选择器，能被IE9及以上以及其他现代浏览器识别

not IE8- .test{background:#ace\9}
