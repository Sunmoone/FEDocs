栅格系统

概述

利用栅格系统将页面布局划分成为多个列，在紧凑的空间中将信息整齐的排布，各个模块移植性强，页面布局清晰规范。

栅格系统示例

以常见的栅格系统为12列举例，形成一个940px宽的容器。
